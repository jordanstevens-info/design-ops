# Create Ticket

Convert design artifacts into a backlog-ready ticket. The ticket describes the
desired product outcome and acceptance criteria without prescribing implementation.

Read:

- `../../references/ticket-work.md`
- `../../references/design-scope.md` only when scope context exists

Use template:

- `../../templates/ticket/jira-ticket.md`

Output the ticket draft, design references used, coverage gaps, and any open
questions that affect acceptance criteria.
