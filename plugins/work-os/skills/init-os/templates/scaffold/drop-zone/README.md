# Drop Zone

Raw incoming material not yet processed or routed.

Use this for quick drops: meeting notes, screenshots, PDFs, links, exports, or
mixed material that needs later classification. Do not store secrets, tokens,
cookies, private config, `.env`, or credentials here.

Agents should treat this as a registered temporary intake surface. Do not flag
every item; call out only items that are stale, promotable, sensitive,
ambiguous, or repeatedly used.
