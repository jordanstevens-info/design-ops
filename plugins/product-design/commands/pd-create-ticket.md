# /pd-create-ticket

Create a backlog-ready ticket from design work.

## Arguments

- `source`: optional branch, prototype, Figma URL, notes, or existing ticket

## Workflow

Route to `product-design-workflow` in ticket mode. Use `.design/`, changelogs, prototypes, Figma, and notes when available.

## Output

An outcome-focused Jira/backlog ticket with user story, design reference, feature specification, acceptance criteria, out of scope, and known constraints.
