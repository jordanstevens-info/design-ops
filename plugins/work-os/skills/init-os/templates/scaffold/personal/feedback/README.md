# Personal Feedback Index

Received feedback, growth themes, and synthesis. Use this for manager, peer,
360, stakeholder, performance-cycle, and self-review feedback when the subject
is personal growth.

Raw source documents stay in `personal/references/hr/`. This index tracks
interpreted signals and themes that may inform goals, tasks, career vision, or
project behavior.

## Lifecycle

Feedback statuses: `captured`, `proposed`, `active`, `applied`, `graduated`,
`archived`, `dropped`, `superseded`.

Applied does not automatically mean archived. Keep applied feedback indexed
while it still shapes growth, goals, tasks, career vision, or project behavior.
Feedback graduates when another artifact owns the behavior, and archives only
when it is no longer load-bearing.

Index sections mean:

- **Active Themes** - active or still-load-bearing applied growth themes.
- **Pending Synthesis** - captured or proposed feedback awaiting interpretation.
- **Recently Applied** - temporary visibility after a change lands.
- **Graduated / Historical** - canonical behavior lives elsewhere now.
- **Dropped / Superseded** - intentionally inactive or replaced feedback.
