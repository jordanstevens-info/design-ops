# /create-preso

Create a design story presentation package.

## Arguments

- `topic`: design work, branch, prototype, Figma URL, research notes, or decision ask

## Workflow

Route to `product-design-workflow` in preso mode. Use `.design/` if present and create `.design/presos/<topic>/` only for repo-backed work.

## Output

A design story package with story, one-pager, deck outline, demo script, assets/links, feedback questions, and impact signals.
