# Sandbox

Messy exploratory work, drafts, experiments, and unrouted thinking.

Use this for active rough work that is more than raw intake but has not earned a
route to `context/plans/`, `context/references/`, `context/resources/`,
`tasks/`, `context/feedback/`, `context/knowledge/`, or a workspace/project
folder.

Agents should treat this as a registered temporary work surface. Do not flag
every item; call out only items that are stale, promotable, sensitive,
ambiguous, or repeatedly used.
